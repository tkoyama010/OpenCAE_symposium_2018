はじめに

GetFEM++は汎用的で効率的な有限要素法のライブラリです。

有限要素法を用いて線形および非線形偏微分方程式を解くためのフレームワークを提供することを目指しています。

あなたがこれから有限要素法プログラムを作成するならばGetFEM++を使用することをおすすめします。

なぜGetFEM++を使用するのか？

  • 弱形式言語

    弱形式を記述する独自の言語を使用して定式化を表現できるため、ユーザーは離散化を行う必要がありません。

  • オブジェクト

    Meshオブジェクト・Femオブジェクト・Integオブジェクト・Modelオブジェクトなど有限要素法の実装に必要なオブジェクトがフレームワークとして実装されています。

  • メッシュ-有限要素法

    メッシュの各凸包に任意の有限要素法を設定することができます。

  • Pythonインターフェース

    Pythonによるプログラミングも可能です。それによりプリ処理・ポスト処理が簡易になっています。

などなど・・・

翻訳作業

GetFEM++のドキュメントにはSphinxが使用されています。 SphinxはPythonのために作成されたドキュメント作成ソフトです。 Sphinxには国際化機能があるため今回はそれを使用して翻訳を行いました。 


import getfem as gf
import numpy as np

m=gf.Mesh('import','gid','../meshes/tripod.GiD.msh')
print('done!')
mfu=gf.MeshFem(m,3) # displacement
mfp=gf.MeshFem(m,1) # pressure
mfd=gf.MeshFem(m,1) # data
mim=gf.MeshIm(m, gf.Integ('IM_TETRAHEDRON(5)'))
degree = 2
linear = False
incompressible = False # ensure that degree > 1 when incompressible is on..

mfu.set_fem(gf.Fem('FEM_PK(3,%d)' % (degree,)))
mfd.set_fem(gf.Fem('FEM_PK(3,0)'))
mfp.set_fem(gf.Fem('FEM_PK_DISCONTINUOUS(3,0)'))

print('nbcvs=%d, nbpts=%d, qdim=%d, fem = %s, nbdof=%d' % \
      (m.nbcvs(), m.nbpts(), mfu.qdim(), mfu.fem()[0].char(), mfu.nbdof()))

P=m.pts()
print('test', P[1,:])

cbot=(abs(P[1,:] + 10) < 1e-6)

pidbot=np.compress(cbot, range(0, m.nbpts()))

ftop=m.faces_from_pid(pidtop)
fbot=m.faces_from_pid(pidbot)
