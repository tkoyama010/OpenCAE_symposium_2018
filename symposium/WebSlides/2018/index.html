はじめに

GetFEM++は汎用的で効率的な有限要素法のライブラリです。 有限要素法を用いて線形および非線形偏微分方程式を解くためのフレームワークを提供することを目指しています。 あなたがこれから有限要素法プログラムを作成するならばGetFEM++を使用することをおすすめします。

なぜGetFEM++を使用するのか？

  • 弱形式言語

    弱形式を記述する独自の言語を使用して定式化を表現できるため、ユーザーは離散化を行う必要がありません。

  • オブジェクト

    Meshオブジェクト・Femオブジェクト・Integオブジェクト・Modelオブジェクトなど有限要素法の実装に必要なオブジェクトがフレームワークとして実装されています。

  • メッシュ-有限要素法

    メッシュの各凸包に任意の有限要素法を設定することができます。

  • Pythonインターフェース

    Pythonによるプログラミングも可能です。それによりプリ処理・ポスト処理が簡易になっています。

などなど・・・

Sphinxの国際化機能

SphinxはPythonのために作成されたドキュメント作成ソフトです。 Sphinxには国際化機能があるため今回はそれを使用して翻訳を行いました。 また、poファイルの管理にTransifexを使用しています。 

                
$ sudo pip install sphinx-intl
$ sudo apt-get install transifex-client

Transifexの利用

前のスライドで作成した*.poファイルをTransifexを使用して管理します。 日本語に翻訳された*.poファイルをダウンロードするには以下のようにします。 APIキーを聞かれた際にはTransifexのアカウントのページでAPIキーを発行してください。 

              
$ sudo aptitude install transifex-client
$ cd /path/to/getfem
$ tx pull -l ja
